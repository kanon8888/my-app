# My Next.js Social Login Project

This is a **Next.js 13+ project** featuring **Email/Password login** and **Google Social Login** using **NextAuth.js**. The project is deployable on **Netlify** and follows best practices for authentication and modern React development.

---

## Features

- Admin Email/Password login
- Google Social Login via **NextAuth.js**
- Protected routes using cookies/session
- Responsive UI using Tailwind CSS
- Deploy-ready for **Netlify**

---

## Technologies Used

- **Next.js 13+** (React Framework)
- **NextAuth.js** (Authentication)
- **Tailwind CSS** (Styling)
- **React Icons** (UI icons)
- **Netlify** (Deployment)
- **JavaScript** (Frontend & Backend logic)
